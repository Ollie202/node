# Miden node

`node` is the main binary for running node infrastructure for a Miden network. It is the binary operators use for
sequencer and full-node operation, and it embeds the store, RPC, and block-producer component crates.

For the overall project map and documentation links, see the [primary README](https://github.com/0xMiden/node#readme).
